# HireTrack ATS

A modern Applicant Tracking System (ATS) that helps recruiters manage jobs, candidates, and hiring workflows.

## Features

- User Authentication
- Dashboard
- Job Management
- Candidate Management
- Application Tracking
- Search & Filters
- Analytics Dashboard

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT Authentication

## Folder Structure

client/
server/
docs/

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
